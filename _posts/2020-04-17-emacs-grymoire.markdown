# Emacs Grymoire

Random tidbits that I need to reference every now and then. Everything is assumed emacs 26.1

# Help / Documentation

* Where are the manuals?
  - C-h r
* Provide documentation for a function
  - C-h f ${FUNCTION}
* Search for commands 
  - C-h a ${KEYWORD} ...
